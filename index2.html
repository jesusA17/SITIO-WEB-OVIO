TEMATICAS

En la presente pagina web se observara la aplicación de la Unidad N°2 del curso de diseños de sitios Web, en la cual se procedera a aplicar cada uno de los conceptos adquiridos.

Las temasticas vsitas en esta unidad son las siguientes: Introducción a HTML5, Estructura básica de HTML5, Aplicación de elementos de HTML5 a un proyecto de diseño Web, Introducción a CSS3, Estructura básica de CSS3, Aplicación de elementos de CSS3 a un proyecto de diseño Web.

OBJETIVOS

OBJETIVOS GENERAL

Crear un objeto virtual de información (Pagina Web) donde se describan los conceptos básicos de seguridad informática, estándares, pautas, políticas y normas en la seguridad de redes.

OBJETIVOS ESPECIFICOS

• Crear una Pagina Web sobre los conceptos básicos de seguridad informática.

• Realizar una investigación sobre los estándares, pautas, políticas y normas en la seguridad de redes

• Diseñar una interfaz amigable que le permita al usuario navegar de manera sencilla sobre mi pagina web

COMPETENCIAS A DESARROLLAR

• El estudiante podrá desarrollar Sitios Web que satisfagan las necesidades del usuario.

• El estudiante podrá operar lenguajes de Diseño web como HTML5 y CSS3.
